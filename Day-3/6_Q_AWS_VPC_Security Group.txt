1 What are security groups?
Ans. Security groups are fundamental to network security; it allows the network traffic in and out of EC2 machines.
	
2 What are inbound rules?
Ans. Inbound rules can be seen as network traffic coming from the internet to EC2 servers.
	
3 What are outbound rules?
Ans. Outbound rules can be seen as network traffic that’s going from EC2 to the internet.
	
4 How to create security groups?
Ans. To create security groups select network, security, select create security group.
