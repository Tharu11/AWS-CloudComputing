1 What is the first thing you need to do to create a VPC?
Ans. The first thing we need to do to create a VPC is select a region.
	
2 What does VPC stand for?
Ans. VPC stands for Virtual Private Cloud.
	
3 How to create VPC?
Ans. VPC can be created by selecting a region, in services go to networking and content delivery select VPC, select VPCs again, select create VPC, give name tag, and IPv4 CIDR block.
	

4 How to create internet gateways?
Ans. To create internet gateways select internet gateways, create internet gateway, give a name, and create.
	

5 How to attach internet gateways to VPC?
Ans. To attach internet gateways to VPC select the internet gateway name, go to action and select attach VPC and select the VPC name.
	

6 How to create subnets?
Ans. To create subnets select subnet, create subnet, subnet name, VPC, Availability Zone, VPC CIDRs, and IPv$ CIDR block and create subnet.
	

7 How to start auto modify auto-assign IP settings?
Ans. To start auto modify auto-assign IP setting select subnet, action tab, click on modify auto design IP settings, click on auto IPv4 checkbox and save it.
	

8 How to create a public/private route table?
Ans. To create a public/private route table select public route table, give a name and select VPC name.
	
9 How to attach a public/private subnet to a route table?
Ans. To attach a public/private subnet to a route table select route table, click on subnet association, edit subnet association,  select a subnet and attach with route table.
