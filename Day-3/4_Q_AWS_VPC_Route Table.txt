1 What is route table?
Ans. A route table contains a set of rules called routes that are used to determine where the network traffic os directed.
	
2 How to create route table?
Ans. To create a route table select create route table and set VPC.
	
3 How to delete route table?
Ans. To delete a route table select delete route table and select / delete.
	
4 How to create internet gateways?
Ans. To create internet gateways find internet gateway, click on internet gateway, give name to internet gateway tag.
	
5 What are internet gateways?
Ans. Internet gateways can be seen as horizontally scaled, highly available gateway that allows communication between instances with our VPC and the internet.
	
6 How to attach internet gateways to VPC?
Ans. To attach internet gateways to VPC click on action, click on attach to VPC, select VPC gateway name.
	
7 How to detach internet gateways to VPC?
Ans. To detach internet gateways to VPC select an internet gateway and click in detach it.
