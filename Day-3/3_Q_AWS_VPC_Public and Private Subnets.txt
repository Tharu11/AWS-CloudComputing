1 What is VPC?
Ans. VPC is a virtual network environment provided by AWS that enables us to launch different AWS resources such as instances, database, load balancers and more.
	
2 VPC stands for?
Ans. VPC stands for Virtual Private Cloud.
	
3 How to create VPC?
Ans. To create VPC in the dashboard look for VPC and click on start VPC and type name and CIDR block and the tendency.
	
4 How to create public/private subnets?
Ans. To create a public/private route table select the public route table, give name to public route table, select a VPC name.
	
5 How to create internet gateways?
Ans. To create internet gateways find internet gateway, click on internet gateway, give name to internet gateway tag.
	
6 How to attach internet gateways to VPC?
Ans. To attach internet gateways to VPC click on action, click on attach to VPC, select VPC gateway name.
	
7 How to create public/private route table?
Ans. To attach a public/private route table select the public route table, give name to the table, select a VPC name.

8 How to create EC2 instances for private/public subnets?
Ans. To create EC2 instances for private/public subnets select instances, click launch instance, select instance and provide information select VPC and subnet, add name.
