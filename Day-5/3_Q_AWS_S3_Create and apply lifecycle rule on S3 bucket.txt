Q1 WHAT IS S3 LIFECYCLE CONFIGURATION?

ANS S3 lifecycle configuration is a cycle that is a time file for S3 bucket and once it’s complete it will delete the S3 bucket.
	

Q2 What are the actions we can configure in S3 lifecycle configuration?

ANS The actions we can configure in S3 lifecycle configuration are transition action, expiration action, and abort incomplete multipart upload.
	

Q3 How to create a lifecycle configuration through aws console?

ANS To create a lifecycle configuration through AWS console go to management, click cycle, add cycle, add name, choose time.
