Q1 WHAT IS OBJECT VERSIONING AND HOW IT IS USEFUL?

ANS Object versioning helps with multiple variant of objects in the same bucket and can be used to preserve, retrieve, and restore every version of every object. It is useful because it is protected from unintended overwrites and deletion.
	

Q2 How to enable versioning using aws console?

ANS To enable versioning using AWS console go to property, versioning, enable versioning.
